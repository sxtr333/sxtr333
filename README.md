### Кейс: Rybnoye Photogrammetry

**Rybnoye Photogrammetry** — проект по подготовке геопространственных материалов для анализа городской территории в рамках благоустройства.

Моя роль в проекте:

* **AI-аналитика**
* **исследование территорий при помощи AI**
* **подготовка данных**
* работа с материалами аэрофотосъёмки
* подготовка аналитической базы для архитектурной команды

Что использовалось в проекте:

* ортофотопланы высокого разрешения
* DSM / цифровая модель поверхности
* 3D mesh / FBX-модели территории
* анализ городской среды и покрытия территории
* подготовка материалов для проектных решений
